PDFtools-py_ PDF 주석 일괄제거, 암호제거

2023-07-31 python version issue


https://github.com/theangkko/PDFtools-py/releases/latest

파이썬, 신규 버전

파일을 하나씩 선택해야 하지만, 오류가 상대적으로 c#버전에 비해 적습니다.


주요기능

  • PDF내 주석을 일괄삭제합니다
  • PDF수정보호(자물쇠) 상태를 해제합니다

사용가능환경)

  • 윈도우
  • MacOS
    Mac에서는.py 소스코드로는 실행가능한데, pyinstaller(실행파일 1개로 만드는) 오류로 독립 실행파일로는 동작하지 않습니다.

사용법)

  1. “Pick File” 버튼을 눌러, 주석제거 또는 암호해제할 pdf 파일을 선택합니다.

  2. 암호해제 기능과 주석제거 기능 중 원하는 기능을 선택합니다.

  3. “Submit” 버튼을 눌러 작업을 실행합니다.



실행파일 다운로드)

위 링크에서 exe파일을 다운받아서 더블클릭합니다.
